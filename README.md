# NEES Lab website

Official website of the NeuroEngineering and Embedded Systems Laboratory at San Francisco State University.

## Pages

- `index.html` — homepage
- `research.html` — research programs
- `people.html` — director, members, and alumni
- `publications.html` — selected publications
- `news.html` — lab news and awards
- `join.html` — prospective student guidance
- `contact.html` — contact information

Shared styles and navigation live in `assets/`. Images migrated from the former ICE Lab website live in `images/`.

## Publishing

Every push to `main` runs `.github/workflows/deploy.yml` and publishes the static site to GitHub Pages. In the repository's **Settings → Pages**, set **Source** to **GitHub Actions**.
